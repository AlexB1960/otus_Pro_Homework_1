# Homework N1

Инструкция запуска тестов страницы https://otus.ru

    Требуемые 3 сценария тестирования реализованы в 4 тестах в тестовом классе CoursesPageTest.
    Запуск всех четырех тестов через консоль возможен как без задания корректирующих параметров (командой mvn clean test), так и со следующими параметрами :
    Для сценария 1 - название курса "Python Developer" задано в переменной currentCourse теста getCorrectCourse. Его можно здесь заменить на другой.
    Для всех сценариев - в командной строке консоли (через аргумент -Dbrowser) можно задать браузер Chrome или Firefox или Edge.
